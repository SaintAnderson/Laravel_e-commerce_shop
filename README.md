Интернет магазин создавался командой из 8ми человек. 

Подготовительные команды:
- composer i 
 - npm i 
Настройки:
- в .env файле прописать подключение к своей базе данных

Выполнить миграции: 
- php artisan migrate 

Заполнить базу информацией: 
- php artisan db:seed 

Запуск проекта: 
- php artisan serve 
- php nmp run dev
